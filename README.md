📊Financial Valuation Analysis
🚀 Project Overview

This project presents a financial valuation analysis of two leading retail companies: Walmart and Target.

The analysis uses financial statement data obtained from official filings submitted to the U.S. Securities and Exchange Commission (SEC). By analyzing these filings, the project estimates the intrinsic value of both companies and compares their financial performance using multiple valuation methods.

The goal is to understand how financial analysts evaluate companies and determine whether they are potentially overvalued or undervalued in the market.

🎯 Project Objectives

The main objectives of this project are:

Analyze the financial performance of Walmart and Target

Extract financial data from SEC Form 10-K filings

Apply financial valuation techniques to estimate company value

Compare valuation outcomes across multiple approaches

Understand how financial data drives investment decisions

📈 Valuation Methods Used
💵 Free Cash Flow (FCF) Valuation

This method estimates the value of a company based on the present value of future free cash flows generated by the business.

Key components include:

Operating Cash Flow

Capital Expenditures

Free Cash Flow projections

Discount rate assumptions

📊 Dividend Growth Model

This model determines company value based on expected future dividend payments and their long-term growth rate.

It incorporates:

Current dividend payments

Expected dividend growth rate

Required rate of return

This method is commonly used for stable companies with consistent dividend distributions.

📉 Market Multiple Approach

The market multiple method estimates company value by comparing financial ratios with market benchmarks.

Examples include:

Price-to-Earnings (P/E) ratio

Revenue multiples

Earnings comparisons

This approach helps evaluate how the market values similar companies.

📂 Financial Data Used

The analysis uses financial statement data extracted from the companies’ Form 10-K annual reports.

Key financial metrics used in the valuation models include:

Total Revenue

Cost of Goods Sold

Operating Income

Net Income

Cash Flow from Operations

Capital Expenditures

Dividend Payments

These financial figures form the foundation of the valuation calculations.
